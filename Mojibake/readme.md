This script inputs SingleLettersRemoved.txt , and changes the character encoding to UTF-8. The new list is saved to  Fixed_UTF8_Letters.txt .

Fixed_UTF8_Letters.txt and SingleLettersRemoved.txt are based on (https://en.wiktionary.org/wiki/User:Matthias_Buchmeier/Spanish_frequency_list-1-5000) licensed under [CC-SA-4.0](https://creativecommons.org/licenses/by-sa/4.0/ ) .
